_vercel-kv key schema
almost:pro:{userId} "1" — user has purchased Pro. Set by /api/unlock-pro after RC confirms entitlement.
almost:gen:{userId}:{branchId}:{templateId} "1" — free-tier generation already consumed for this (branch, template) combo.
almost:branch:{userId} First branchId used by this free-tier user. Prevents branch-switching without Pro.
